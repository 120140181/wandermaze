# Wandermaze

## 🎮 Game Overview
Wandermaze is a 2D top-down platformer game filled with illusions and hidden traps. Players navigate through a puzzle-filled world where every movement decision could lead to safety or unexpected traps.

## 🌟 Features
- **Puzzle-based Levels**: Each level is designed with traps, dead ends, and hidden teleporters.
- **Illusory Directions**: Players will encounter misleading directional hints, such as compass symbols that lead them astray.
- **Dynamic Level Design**: The layout of levels can change based on player actions, creating a dynamic gameplay experience.
- **Checkpoint System**: Hidden checkpoints that may or may not be active, adding an element of surprise.
- **Humorous Animations**: Fun animations play when the player gets trapped, making failure entertaining.

## 🚀 Getting Started
1. **Clone the Repository**:
   ```
   git clone https://github.com/yourusername/wandermaze.git
   cd wandermaze
   ```

2. **Install Dependencies**:
   Make sure you have Python installed, then run:
   ```
   pip install -r requirements.txt
   ```

3. **Run the Game**:
   Execute the main script:
   ```
   python src/main.py
   ```
